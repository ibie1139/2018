Suyesh's way to create a React App:
1.create a folder
2.CMD:
1)npm init -y
2)npm install --save react react-dom
3)npm install --save next
(this is called next.js)
4)code .
5)npm run start

3.create a folder called 'pages' in your project root
4.change "next" to "next dev" in package.json, meaning development environment
5.CMD:
npm run start 
(Now, things are compiled, nd the localhost is ready)


**If need to change something, press CRTC+C
